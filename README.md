QR Code Generator

A lightweight, single-file web application that generates downloadable QR codes from URLs or text. It features color customization, custom titles, and high-resolution PNG downloads.

üöÄ Features

Instant Generation: Generates QR codes in real-time as you type.

Customization: Choose custom foreground and background colors.

Title Embedding: Optionally add a title/label above the QR code (perfect for "WiFi Password", "Menu", etc.).

Smart Download: Composites the title and QR code into a single PNG image for easy printing.

Responsive Design: Works beautifully on desktop and mobile devices.
